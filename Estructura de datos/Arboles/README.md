# Gestor de Números con Árbol Binario de Búsqueda (BST)

## Descripción
Aplicación de consola en C# que implementa un Árbol Binario de Búsqueda (BST) con operaciones básicas de inserción, búsqueda, eliminación y recorridos.

## Características
- ✅ Inserción de números en el BST
- ✅ Búsqueda de números con visualización de ruta
- ✅ Eliminación de números (todos los casos: hoja, un hijo, dos hijos)
- ✅ Recorridos: Inorden, Preorden, Postorden
- ✅ Cálculo de altura y número de nodos
- ✅ Exportación del recorrido inorden a archivo
- ✅ Persistencia de datos (guardar/cargar desde archivo)
- ✅ Interfaz de consola interactiva
- ✅ Casos de prueba integrados

## Requisitos
- .NET 8.0 SDK o superior
